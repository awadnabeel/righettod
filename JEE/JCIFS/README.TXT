This project have to objective to show samples of use of the Java JCIFS API. 
The context is a connection to a share (with authentication or not) and download/upload of files.

JCIFS is an Open Source client library that implements the CIFS/SMB networking protocol in 100% Java. 
CIFS is the standard file sharing protocol on the Microsoft Windows platform (e.g. Map Network Drive ...). 
This client is used extensively in production on large Intranets. 

http://jcifs.samba.org/
