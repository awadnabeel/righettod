** Description

This project contains a proof of concept in order to create an counter measure that can
be sent to an person who is profiling your web application. This counter measure
grab informations from local computer in order to try to clearly identity/locate  the person.

This POC is linked to OWASP wiki article "https://www.owasp.org/index.php/Detect_profiling_phase_into_web_application", see project "ProfilingDetectionPOC" in this repository.

** Prerequisite

Maven 3.x:
http://maven.apache.org/

JavaSE JDK 1.7.x:
http://www.oracle.com/technetwork/java/javase/downloads/index.html

** Build binaries

Using a shell, run command "mvn clean package".
