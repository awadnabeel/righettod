This project have to goal to show a sample of integration/communication between a JavaFX frontend and a Java backend,
the sample is a client that get the Google Talk account buddies from the GTalk server using XMPP protocol

The XMPP Api used by the backend (that is a simple pojo) is SMACK
http://www.igniterealtime.org/projects/smack/index.jsp

The IDE used is NetBeans 6.5 with the JavaFX plugins
http://www.netbeans.org/

Some JavaFX links
Language reference : http://openjfx.java.sun.com/current-build/doc/reference/JavaFXReference.html
API Javadoc        : http://java.sun.com/javafx/1/docs/api/
Learning JFX       : http://java.sun.com/javafx/1/tutorials/core/index.html

Others link about XMPP & GTalk
http://abhijeetmaharana.com/blog/2007/10/28/writing-a-gtalk-jabberxmpp-client/comment-page-2/
