** Introduction
This project is a small web application providing a relay to download file, you can use it when your
web proxy do not allow you to download directly the desired file....

This application is developed under Eclipse Galileo with the Google App Engine Eclipse plugin
is deployed on Google App Engine.

** Links
Google App Engine for Java : http://code.google.com/appengine/docs/java/gettingstarted/
Application url on GAE     : http://filedownloadrelay.appspot.com/
