This project have to objective to show samples of use of the JAXB XML/Object mapping API using the implementation
provided by the JavaSE6 JDK.

The Eclipse plugin below is used to generate Java annoted classes from XSD schema file :
https://jaxb-workshop.dev.java.net/plugins/eclipse/xjc-plugin.html

A JavaSE6 JDK is required to run this project !
