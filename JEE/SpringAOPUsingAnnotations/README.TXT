***********************************************
* Spring AOP Sample project using annotations *
***********************************************

This sample use "SpringAOP Framework" instead of "full AspectJ Framework" 
(ie only a part of AspectJ functionalities is supported by Spring and this one compose 
the 'SpringAOP' Framework) and use AspectJ annotations style for aspects configuration
(see point "6.1.2. Spring AOP capabilities and goals" for explanation between frameworks and configuration
choice when use AOP in Spring)

This sample is based on tutorial available on :
http://static.springframework.org/spring/docs/2.0.x/reference/aop.html
