This project have to goal to show a sample of use of JAX-RS, a new REST web services API,
through the reference implementation named "JERSEY" (https://jersey.dev.java.net/)

This project use :
- Maven2 for the project struture/build
- NetBeans 6.5 for the IDE
- Glassfish V3 Prelude for the AS
