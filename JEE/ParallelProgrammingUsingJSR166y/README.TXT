This project have to objective to show how use parallel programming in java using the features provided by the JSR166y. This JSR,

API Javadocs        : http://gee.cs.oswego.edu/dl/jsr166/dist/jsr166ydocs/
Samples             : http://artisans-serverintellect-com.si-eioswww6.com/default.asp?W1

This project require JavaSE 6 to compile and run...
