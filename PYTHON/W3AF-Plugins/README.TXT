* README *
**********

This project contains W3AF (http://w3af.sourceforge.net) custom plugins.

The project is in Aptana Studio + PyDev format.

Use "deploy.py" script to deploy plugins to your local W3AF instance.

This project is used to store plugins that are under developement by me or 
pending validation from W3AF project leader in order to be integrated 
into an W3AF official release...
